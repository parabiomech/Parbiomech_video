# 🎿 Parbiomech Video Analysis - Ski Edition

**MediaPipe 기반 스키 동작 분석 시스템**

## ✨ 주요 기능

### 📊 3가지 분석 모드

1. **🎯 시점 분석**: 특정 시점의 포즈와 각도 분석
2. **📈 구간 분석**: 시작~끝 구간의 관절 궤적 그래프 + 스트로모션 오버레이
3. **🎿 스키 분석**: 5-line 시스템 기반 스키 기술 분석 + 스트로모션

### 🎨 핵심 기능

- 📹 **비디오 업로드**: 드래그 앤 드롭 지원 (MP4, AVI, MOV, WebM)
- 🏷️ **시점 태그**: 분석할 특정 시점 선택 (스페이스바 단축키)
- 🎯 **포즈 감지**: MediaPipe.js로 33개 키포인트 실시간 추적
- 📊 **종합 각도 분석**: 
  - 절대각도: 분절의 수평면 대비 기울기 (머리, 몸통, 대퇴, 하퇴, 발)
  - 상대각도: 관절의 굴곡/신전 각도 (목, 고관절, 무릎, 발목)
- 🎨 **스트로모션**: 구간 내 모든 프레임을 반투명 오버레이로 시각화
- 📈 **궤적 그래프**: 관절 이동 경로를 부드러운 곡선으로 표시 (Chart.js)
- 🎿 **5-Line 시스템**: 스키 턴 분석을 위한 특화 시각화

## 🚀 사용 방법

### 시점 분석 모드
1. **비디오 업로드**: "📁 파일 선택" 버튼 클릭 또는 드래그 앤 드롭
2. **시점 태그**: 영상을 재생하며 분석할 시점에서 **스페이스바** 누르기
3. **분석 시작**: "🔍 분석 시작" 클릭
4. **결과 확인**: 각 시점의 포즈 이미지와 각도 데이터 확인

### 구간 분석 모드
1. **구간 시작**: 분석할 구간의 시작점에서 **스페이스바** (파란색 태그)
2. **구간 끝**: 끝점에서 **스페이스바** (빨간색 태그)
3. **분석 시작**: "📈 구간 분석 시작" 클릭
4. **결과 확인**: 관절 궤적 그래프 + 스트로모션 오버레이

### 스키 분석 모드
1. **구간 설정**: 턴 시작~끝 지점 태그
2. **분석 시작**: "🎿 스키 분석 시작" 클릭
3. **결과 확인**: 5-line 시각화 + 각도 그래프 + 스트로모션

## 💡 기술 스택

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **AI Model**: MediaPipe Pose (Google)
- **시각화**: Chart.js (궤적 그래프), Canvas API (스트로모션)
- **라이브러리**: MediaPipe.js (CDN), Chart.js (CDN)
- **호스팅**: GitHub Pages

## 🎨 특징

- ✅ **완전 무료**: 서버 비용 없음
- ✅ **설치 불필요**: 브라우저만 있으면 실행
- ✅ **Python 불필요**: 순수 JavaScript로 구현
- ✅ **크로스 플랫폼**: PC, Mac, 모바일 모두 지원
- ✅ **실시간 분석**: MediaPipe 기반 고속 포즈 추정
- ✅ **스트로모션 시각화**: 구간 내 모든 동작을 한눈에
- ✅ **부드러운 궤적**: Cubic Bezier 보간으로 자연스러운 곡선
- ✅ **스키 전문 분석**: 5-line 시스템 + 턴 각도 분석

## 🎿 스키 분석 세부 기능

### 5-Line 시스템
- **Line 1**: 어깨 중심
- **Line 2**: 고관절 중심
- **Line 3**: 무릎 중심
- **Line 4**: 발목 중심
- **Line 5**: 발 중심

### 각도 측정
- **상체 전경각**: 몸통의 수직 대비 전방 기울기
- **대퇴 각도**: 대퇴부 세그먼트 각도
- **하퇴 각도**: 하퇴부 세그먼트 각도
- **무릎 굴곡각**: 고관절-무릎-발목 사이 각도
